objLayerAnimation

prtAlwaysActive > objLayerAnimation

Description

Switches certain tile layers on / off to create an animation effect. This only works for tiles on a 16x16 grid.

Variables

NameValueDescription
animationLayer1

Layer the animated tiles start on. The later animation frames have increasing depth from this, incrementing by 1.

animLength1

How many frame the animation is.

animateOnTransitionfalse

If the animation plays on screen transitions or not.

timer0
tileID0
