objObjectSetter

objects / Collision / objObjectSetter
This object has no parent.

Description

Autotiler for tilesets, place the whole tileset image somewhere in the level and place this

object on the top left tile of the tileset, then place whatever object you want over the tiles.

Once the level starts all the tiles of the tileset in the level that are in the same layer as the object setter will have

an instance of the object you placed over it before.
